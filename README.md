# Vehicle Fuel Efficiency Analysis

## Project Overview
This project performs exploratory data analysis (EDA) on the Auto MPG dataset to understand the factors that influence vehicle fuel efficiency.

The analysis explores how vehicle characteristics such as weight, acceleration, model year, and country of origin affect miles per gallon (MPG).

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Jupyter Notebook

## Dataset
The dataset contains historical information about vehicles including:

- MPG (miles per gallon)
- Cylinders
- Displacement
- Horsepower
- Vehicle weight
- Acceleration
- Model year
- Country of origin

## Key Analysis
The notebook includes several visualizations to explore fuel efficiency:

1. Average fuel economy by country of origin
2. Fuel efficiency trends over time
3. Vehicle weight vs fuel efficiency
4. Vehicle acceleration vs fuel efficiency

## Key Insights
- Vehicle weight shows a strong negative relationship with fuel efficiency.
- Fuel efficiency improves over time, suggesting technological advancements in automotive design.
- Acceleration shows a weaker relationship with MPG compared to weight.
- Vehicles from different regions display variations in average fuel economy.

## Conclusion
The analysis highlights that vehicle design characteristics, particularly weight, play a significant role in determining fuel efficiency. Improvements over time suggest ongoing technological progress in the automotive industry aimed at improving fuel economy.
